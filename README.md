# Walletfy
Library and wrapper for the creation and consultation of blockchains and wallets.

## What can I do with this library?
- [x] Create wallets.
- [x] Obtain the balance of a wallet.
- [ ] Make transfers.
- [ ] Detect transfers in real time.
